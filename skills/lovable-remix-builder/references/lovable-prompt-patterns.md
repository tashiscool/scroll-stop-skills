# Lovable Prompt Patterns

Use these prompt patterns to build section by section.

## Initial Hero Prompt

```text
Create the homepage hero exactly in this visual direction. Keep it premium, responsive, and polished. Use a clear headline, one primary CTA, one secondary CTA, and a product demo area. Match the dark/light mood, typography feel, spacing, and premium visual hierarchy from the reference.
```

## Add Navbar Prompt

```text
Add a navbar above the hero. Make sure to use the same styles, colors, fonts, and spacing language already established in the hero section. Keep the mobile menu polished and responsive.
```

## Add Section Prompt

```text
Add a new section below the existing content using this structure. Preserve the exact design language already on the page: same colors, same type scale, same border treatment, same spacing rhythm, and the same level of premium polish. Make sure the section also looks great on mobile.
```

## Replace Hero Image With Video Prompt

```text
Replace the hero image with this hosted video. Keep the same framing and premium feel. The video should autoplay, be muted, loop, show no controls, and fit edge to edge cleanly inside the hero media area.
```

## Background Video Prompt

```text
Add a background video layer to this section using the provided video URL. Keep it autoplay, muted, looping, and without controls. Stretch it edge to edge using absolute inset positioning while preserving text readability with a subtle overlay if needed.
```

## Rewrite Prompt

```text
Rewrite all homepage content to be about [target brand / audience / niche]. Keep the same structure and section order, but replace the content, labels, icons, and CTA copy so it feels purpose-built for that business.
```

## Secondary Page Prompt

```text
Create an [About / Pricing / Contact / Features] page following the styles from the homepage. Keep the same design language, spacing, typography, button styles, and premium tone. This page should feel like part of the same product system.
```

## Mobile QA Prompt

```text
Audit the current page for mobile responsiveness. Fix any broken spacing, oversized typography, awkward wrapping, overlapping media, or weak menu behavior. Do not change the design language, only improve the responsiveness and polish.
```
