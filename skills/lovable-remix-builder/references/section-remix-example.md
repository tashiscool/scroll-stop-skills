# Section Remix Example

This is the pattern to follow when rebuilding a site from references:

1. pick the best hero reference
2. build the hero
3. add navbar
4. add features
5. add CTA
6. rewrite content
7. add secondary pages

## Example Flow

User goal:
```text
Build a church website in Lovable with a premium dark style based on this SaaS reference.
```

Agent response strategy:

### Step 1
Choose the conversion-worthy parts of the reference:
- hero with strong headline
- feature grid
- testimonial / trust block
- CTA

### Step 2
Generate the first hero prompt:
```text
Create the homepage hero exactly in this style direction. Dark premium background, clean typography, left-aligned headline, supporting copy, two buttons, and a hero media area. Keep it fully responsive for desktop, tablet, and mobile.
```

### Step 3
Add the navbar:
```text
Add a navbar above the hero using the same colors, fonts, button styling, and spacing language already established. Keep the mobile menu polished.
```

### Step 4
Add a feature section:
```text
Add a new feature section below the hero using a premium three-column card layout. Match the existing styles exactly. Keep it mobile responsive.
```

### Step 5
Rewrite the content:
```text
Rewrite all homepage content to be about a Christian church in Vietnam. Keep the same structure and design language, but replace all text, labels, and CTA copy for that audience.
```

### Step 6
Create a secondary page:
```text
Create an About Us page following the styles from the homepage.
```
