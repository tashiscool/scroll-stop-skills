# Reference Selection

Choose references that are not only attractive but structurally useful.

## What Good References Have

- A strong, clear headline
- A visible primary CTA
- A convincing demo visual or proof block
- A limited number of visual ideas that can actually be reproduced
- Mobile-friendly section logic

## What To Look For

### Hero
- clear headline
- supporting description
- one or two buttons
- demo image or short video

### Features
- scannable cards or columns
- consistent icon style
- clean spacing

### Social Proof
- logo strip
- testimonial card
- proof of trust or adoption

### CTA
- one direct action
- simple layout
- high contrast

## What To Avoid

- references with no obvious CTA
- references that are mostly decorative
- references that depend on overly bespoke art direction for every section
- references that collapse on mobile

## Default Advice

If a reference looks amazing but does not communicate clearly in 3 seconds, do not use it as the main reference.
