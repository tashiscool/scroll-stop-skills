# Lovable Remix Builder

Use the installed `lovable-remix-builder` skill from `~/.claude/skills/lovable-remix-builder`.

## Goal
Turn reference websites into a premium Lovable build plan with section-by-section prompts, consistent styling, content rewrites, and follow-on pages.

## Workflow

1. Pick the best references and identify the high-converting sections.
2. Build the homepage one section at a time.
3. Keep Lovable aligned to the same design language throughout.
4. Write copy that explains the value add, mechanism, and practical payoff instead of just listing features.
5. Rewrite the site for the target brand or niche.
6. Create secondary pages in the same style.
7. Check mobile after each major addition.

## Output

- a section plan
- the Lovable prompts in order
- any rewrite prompts
- any secondary-page prompts
- a short mobile QA checklist
