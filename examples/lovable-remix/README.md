# Lovable Remix Example

This folder documents the intended usage pattern for `lovable-remix-builder`.

Use the skill when you want to:
- choose good references
- break them into reusable sections
- recreate those sections in Lovable
- rewrite the content for a different company, niche, or client
- generate follow-on pages using the same design system

Suggested starter prompt:

```text
Use $lovable-remix-builder to turn this premium SaaS reference into a section-by-section Lovable build for a dating app. Keep the dark polished feel, rewrite the content for our product, and generate the homepage plus an About page.
```
